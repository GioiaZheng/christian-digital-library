# 系统架构

## 公开层

GitHub 保存网站代码、公开书目、文档和测试。GitHub Pages 提供静态搜索、分类和书目详情页。

## 文件层

图书文件与公开仓库分开保存。静态网站不包含私有文件地址，也不直接生成下载链接。

## 生成流程

```text
公开书目 + 分类
      │
      ▼
 静态页面生成器
      │
      ▼
 GitHub Pages
```

## 预览与受限访问

```text
详情页
  ├─ 显示前 5 页预览图
  └─ 下载 / 全文阅读入口
        │
        ▼
  Cloudflare Worker 校验密码
        │
        ▼
  读取 R2 私有对象并返回临时访问结果
```

- 预览图是单独生成的公开资源，不等同于完整文件。
- 封面图使用第一页单独生成的图片，和原始 ZIP/PDF 分开保存。
- 下载和阅读全文不由 GitHub Pages 直接处理。
- 密码校验、临时链接和 R2 读取都放在 Cloudflare Worker。
- 前端页面不保存密码，也不暴露 R2 原始对象地址。

## 安全边界

- 公开仓库中的内容都视为可被任何人读取。
- 只提交网站运行所需的公开数据。
- 密钥、内部记录和私有文件信息不进入公开仓库。
- 文件访问控制由未来的服务端实现。
